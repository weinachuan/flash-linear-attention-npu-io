状态、风险与优先级规则

以任务计划结束日作为 DDL；北京时间 DDL 当天 24:00 后进入延期判断。

状态

  • delay超过 DDL，且交付件未完备。
  • done交付件完备。
  • doing未超过 DDL，且已规定起始闭环时间、正在推进。
  • todo待排人力，或只有事项但未规定闭环时间。
  • Pending更高优先级事项插入，导致任务因人力问题挂起。

风险

  • 低PR 已合入；或有未合入 PR 且距离 DDL 大于 3 个工作日。
  • 中有未合入 PR 且距离 DDL 小于等于 3 个工作日；或无 PR 但距离 DDL 大于 6 个工作日。
  • 高无 PR 且距离 DDL 小于等于 6 个工作日；或责任人为待排人力/待填写。
  • 工作日按中国法定节假日、调休工作日和周末计算。

优先级

  • P0有对外承诺的事项。
  • P1无对外承诺但收益较高、涉及易用性，或内部目标早于对外交付承诺。
  • P2不阻塞使用、收益较小、自规划类型事项。
